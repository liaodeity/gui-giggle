## 关于 Gui Giggle

Gui Giggle 是一个laravel的前后端后台系统，可以帮你快速的搭建一个后台系统:

- 演示站点：

简单不复杂，让你无文档，只要会laravel就能正常开发

### 相关功能

- 用户管理
- 账号管理
- 角色管理
- 权限管理
- 日志管理
- 登录管理
- 配置管理
- 菜单管理
- 版本更新
